PHPLOC Ansible role
=========

If you are a PHP developer and you use PHPLOC.

Requirements
------------
  - `php ^5.6 || ^7.0` 

Role Variables
--------------
None

Dependencies
------------
None

Example Playbook
----------------
    - hosts: php
      roles:
        - devsoul.phploc

Check if it's installed:
------
    phploc --version
